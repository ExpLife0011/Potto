§ CreateInstance()

POTTO_API POTTO_ERROR Potto::CreateInstance ( const PottoUuid &  classId,
const PottoUuid &  interfaceId,
void **  ppv,
const std::string &  moduleName = std::string() 
)

Creates the instance.

Parameters
classIdThe class id.
interfaceIdThe interface id.
ppvThe pointer to receive the interface.
moduleNameOptional. The path of the module to use.
Returns
POTTO_E_OK if successful.

If the moduleName is not empty, this function will try to find the module in the module entry list. If the module is found the function will use the module found, if it is not found then the function will load the module. Then the function will create the instance from the the module. If the module is not found or loaded or failed to create the instance the function will fail with the error code.
